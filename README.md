**注意事项：**
1.  工具依赖 `MongoDB` ，使用前请填写 `地址` 、 `数据库` 、 `集合` 。
2.  使用前请配置玩家数据目录（以 `\` 结尾），正版记得 `重新解析UUID映射表`  ，盗版自行按照格式填写 `uuid2name.json`
